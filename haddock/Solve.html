solveTheGame :: String -> Int -> [String] -> [String] -> IO () #

Función que maneja el flujo del solver, muestra la palabra sugerida, y luego pide la pista. Si la pista no es completamente buena, procede a buscar una palabra a sugerir y se repite el proceso. Si se llega a más 6 turnos. Finaliza el programa.
